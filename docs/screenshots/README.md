# Ekran Görüntüleri

Bu dizin, README dosyalarında kullanılacak ekran görüntülerini içerir.

## Gerekli Ekran Görüntüleri

Aşağıdaki ekran görüntülerini ekleyin ve README.md ve README.en.md dosyalarındaki TODO yorumlarını güncelleyin:

1. **homepage.png** - Ana sayfa
2. **dashboard.png** - Öğrenci dashboard
3. **test-screen.png** - Test çözme ekranı
4. **assignment-page.png** - Ödev sayfası
5. **progress-dashboard.png** - İlerleme dashboard

## Ekran Görüntüsü Ekleme Adımları

1. Ekran görüntüsünü bu dizine ekleyin
2. README.md dosyasında ilgili TODO yorumunu bulun
3. `<!-- TODO: Ana sayfa ekran görüntüsü ekle -->` yerine şunu ekleyin:
   
   ```markdown
   ![Ana Sayfa](docs/screenshots/homepage.png)
   ```

4. README.en.md için aynı işlemi İngilizce açıklamalarla yapın
